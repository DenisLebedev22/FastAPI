# Start application

Чтобы запустить приложение на FastAPI введите в командную строку "docker-compose up" для обычного запуска приложения находясь в директории /test
(данная команда запустит докер композ, в который включен докер контейнер с FastAPI and nginx)
Чтобы запустить приложение имея возможность продолжить работу с сервером и строкой - пропишите "docker-compose up -d" 
or "docker-compose up -d --build", чтобы пересобрать сборку и запустить - используйте при изменении конфигов Dockerfile docker-compose, файлов приложения

# Данное приложение представляет из себя:
main.py - само приложение на FastAPI
Dockerfile - python:3.14 - докер контейнер на пайтон запускающий приложеня FastAPI
docker-compose.yml - докер композ запускающий два контейнера с FastAPI and nginx
